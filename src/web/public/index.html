Habilidades Autónomas (Tools)

Las habilidades activas permiten al Agente ejecutar código de forma autónoma durante el razonamiento.

Herramientas Nativas

Puedes añadir más desarrollando archivos en `src/agent/tools`. Estas son las detectadas actualmente:

Obtener Hora Local (get_current_time)
Otorga percepción temporal al LLM nativamente para programar o saber qué día es.
Leer Archivo (read_file_local)
Lectura de archivos alojados en el mismo contenedor donde está este Agente.
Escribir Archivo (write_file_local)
Escritura o alteración del código y contenido de archivos locales.
Listar Directorio (list_dir_local)
Inspección del árbol de archivos en base a una ruta local dada.
Ejecutar Shell/Bash Local (run_shell_local)
Permite al Agente lanzar scripts npm, bash_commands o comandos locales autónomos.
Túnel SSH Remoto (run_ssh_command)
Conexión y ejecución de comandos SSH en Host externos usando sus credenciales seguras.
